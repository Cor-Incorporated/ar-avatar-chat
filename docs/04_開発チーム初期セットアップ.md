### 4. srcディレクトリの準備

**指示**: 開発作業用のサブディレクトリを作成してください。

```bash
cd /Users/teradakousuke/Developer/ar-avatar-chat/src

# └── assets/
#     ├── markers/

#     └── animations/
```
